The Code

You've seen the paint & wine classes, now it's time to get techy and learn web development basics over trendy cocktails.
The first thing to know is no prior programming experience is required! We can say that because we have plenty of mentors on hand to help you through the tough parts, no matter when you choose to sign up.
As an intro to programming, you'll get an overview of how sites are built for the web, from basic HTML pages to building single page applications. You will learn about version control and how to collaborate with other developers using GitHub. We like to think of it as the essential web developer tool kit.
When you've got the basics down, you'll practice creating and deploying your own projects using resources like yeoman to show off your epic coding prowess. And once you've hit the Ballmer Peak courtsey of crafted cocktails, it's to infinity and beyond in writing code and learning to mentor others.
